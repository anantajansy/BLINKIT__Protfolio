# Blinkit Sales Analysis Dashboard (Power BI)

## Project Overview

This project analyzes Blinkit sales data using Power BI to understand product performance, outlet trends, and customer ratings.

The dashboard provides insights into:

* Total Sales
* Average Sales
* Number of Items
* Average Customer Rating
* Sales by Item Type
* Sales by Outlet Location
* Sales by Outlet Size
* Sales trends over years

## Tools Used

* Power BI
* Data Modelling
* DAX
* Data Visualization

## Key Insights

* Total Sales reached **$1.20M**
* **Supermarket Type 1** generated the highest sales
* **Tier 3 locations** contributed the most revenue
* Fruits, Snacks, and Household items generated high sales
* Average rating across products is **3.9**
